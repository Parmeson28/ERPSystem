# PORTO Technical ERP
Porto Technical ERP is a desktop ERP system designed to assist workshop technicians in managing stock and quickly accessing technical drawings relevant to their daily tasks.
This project was built as an independent study to explore ERP system architecture, improve Java/JavaFX development skills, and strengthen SQL knowledge using PostgreSQL.

## Tech
Language: Java
UI framework: JavaFX
DB Connection: JDBC
Database: PostgreSQL
Build tool: Gradle

## How to run it
To run the project on your own PC, all you need is to download Java and open the project on your IDE of preference. I personaly use VS code, so I will be showing how to run on VS code.

Step 1: Import the project into the IDE of your preference

Step 2: Create a .env file inside the "App" folder

Step 3: Inside the .env file you must pass three variables -> DB_URL / DB_USER / DB_PASSWORD

Step 4: Run the code on the terminal through the command .\gradlew run

If you did all the steps correctly, the program should run properly and you should see the main screen of the app:
